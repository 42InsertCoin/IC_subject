# Pull Request 사용법

## 사용법
1. 원본 저장소 fork (현재 보고 있는 저장소)
2. 원하는 브랜치에 작업 후 commit과 push를 한다. (현재는 master 브랜치만 존재)
3. push 이후 본인이 fork한 페이지에서 pull-request 생성
    * 이 때 본인이 fork한 저장소에서 현재 IC_subject 저장소로 pull-request를 생성해야함
